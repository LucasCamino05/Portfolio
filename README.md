# Portfolio — Lucas Camino

Monorepo del portfolio personal.

```
.
├── frontend/   Angular 21 — UI del portfolio (Netlify)
└── backend/    Spring Boot — API del formulario de contacto (Render)
```

## Frontend

Portfolio single-page bilingüe con tema dual y animaciones.
Ver [`frontend/README.md`](./frontend/README.md) para detalle.

```bash
cd frontend
npm install
npm start
```

## Backend

Endpoint `/contact` (envío de email) y `/health` (warm-up).
Pendiente de inicializar.

## Deploy

- **Frontend** → Netlify, deploy automático desde GitHub.
- **Backend** → Render free tier (con warm-up al cargar el front para mitigar cold start).
